# 📊 Personal Budget Tracker

**Personal Budget Tracker** — веб-приложение для спокойного и наглядного управления личным бюджетом.  
Проект ориентирован на ежедневное использование: быстрый журнал операций, аналитика по месяцам, локальная работа без сети и синхронизация через аккаунт.

🔗 **Официальный доступ к веб-приложению:**  
👉 [https://bonaqu.github.io/personal-budget-tracker-ru/](https://bonaqu.github.io/personal-budget-tracker-ru/)

> ⚠️ Веб-приложение предназначено для использования **через официальную страницу GitHub Pages проекта**.  
> Локальный запуск, клонирование и модификации запрещены без разрешения автора.

---

## ✨ Ключевые возможности

### 💰 Управление бюджетом
- учет доходов и расходов по месяцам
- отдельные блоки для доходов, долговых обязательств, регулярных платежей, текущих расходов и хотелок
- шаблоны, избранное и быстрые сценарии
- ручной и автоматический стартовый остаток месяца
- лимиты по категориям и цели накопления

### 📈 Аналитика и визуализация
- динамика баланса по дням
- структура расходов по категориям
- календарь трат и тепловая карта расходов
- быстрый срез месяца с ключевыми метриками

### 🌐 Локальная работа и облачная синхронизация
- приложение продолжает работать офлайн
- локальные изменения сохраняются на устройстве
- при входе в аккаунт бюджет синхронизируется с облаком
- demo-сессия и реальный аккаунт изолированы друг от друга

### 📱 Адаптация под экраны
- отдельный mobile UX для узких экранов
- desktop и mobile сценарии проверяются отдельно
- интерфейс адаптирован под популярные телефонные и desktop-разрешения

---

## 🧠 Архитектура

- фронтенд: статическое SPA
- облачный API: **Cloudflare Worker**
- облачное хранилище: **GitHub Gist (JSON)**
- локальное хранилище: `localStorage` для локальной сессии, очереди изменений и временного состояния синхронизации

---

## 💾 Резервные копии

- доступен экспорт полного backup в JSON
- доступен импорт backup из JSON
- при импорте в авторизованной учетке данные не теряются и затем синхронизируются с облаком
- при roundtrip-проверке сохраняются:
  - стартовый остаток месяца
  - режим `Авто / Ручной ввод`
  - категории операций
  - шаблоны и прочие ключевые поля

---

## 🔄 Синхронизация

Приложение показывает понятные статусы:
- локально
- синхронизация
- синхронизировано
- нет связи
- ошибка синхронизации

В окне **Сессия и синхронизация** отображаются:
- главный источник данных
- наличие локальных изменений
- последняя синхронизация

---

## 🧪 Надежность

Проект ориентирован на стабильную ежедневную работу:
- корректное разделение demo-режима и реального аккаунта
- устойчивый import/export резервных копий
- совместимость со старыми backup-файлами
- явные статусы синхронизации и безопасный offline-first сценарий

---

## 📥 Импорт / 📤 Экспорт

- экспорт создает JSON-файл с полным состоянием бюджета
- импорт восстанавливает данные из backup
- legacy backup-файлы поддерживаются и нормализуются автоматически

---

## 🔐 Безопасность

- авторизация и регистрация доступны только онлайн
- demo-режим не смешивается с данными реального аккаунта
- токены проверяются сервером и инвалидируются после выхода
- критические проверки синхронизации выполняются на стороне Worker

---

## 🚀 Как пользоваться

1. Перейдите по ссылке:  
   👉 [https://bonaqu.github.io/personal-budget-tracker-ru/](https://bonaqu.github.io/personal-budget-tracker-ru/)
2. Войдите в demo или создайте аккаунт
3. Ведите бюджет в браузере
4. При необходимости экспортируйте backup

Никаких установок, расширений или отдельного сервера не требуется.
