# 📊 Personal Budget Tracker

**Personal Budget Tracker** — это  веб-приложение для осознанного управления личными финансами.  
Проект ориентирован на надёжность и удобство повседневного использования.

🔗 **Официальный доступ к веб-приложению:**  
👉 https://bonaqu.github.io/personal-budget-tracker-ru/

> ⚠️ Веб-приложение предназначено для использования **только через GitHub Pages** проекта.  
> Локальный запуск, клонирование и модификации запрещены без разрешения автора.

---

## ✨ Ключевые возможности

### 💰 Управление бюджетом
- Учёт доходов и расходов
- Категории и подкатегории
- Поддержка повторяющихся платежей и шаблонов
- Быстрые операции (quick actions)

### 📈 Аналитика и визуализация
- Динамика баланса по дням
- Структура расходов по категориям
- **Вертикальный маркер текущего дня** на графике баланса
- **Cashflow Heatmap** — мини-календарь трат по дням месяца

### 🧠 Финансовый ассистент
Ассистент анализирует данные и даёт ненавязчивые рекомендации:
- Burn rate (скорость расходования средств)
- Аномальные категории
- Потенциальные кассовые разрывы
- Поведенческие паттерны трат

### 🌐 Offline-first + Sync
- Веб-приложение продолжает свою работу даже при **отвале интернета**!
- Все изменения в оффлайне сохраняются в "очередь" для последующей синхронизации в облаке при восстановлении сети
- Ничего не теряется

### 🔄 Состояния синхронизации
Явно отображаются:
- `ONLINE`
- `OFFLINE`
- `RECONNECTING`
- `SYNCED`

Также показывается:
> “Синхронизировано X минут назад”

---

## 🧠 Архитектура (кратко)

- 📄 **Один HTML-файл** — весь фронтенд
- ☁️ **Cloudflare Worker** — мост
- 🗂 **GitHub Gist (JSON)** — как облачное хранилище данных
- 🔄 Все сетевые операции выполняются через `fetch` к Worker API

---

## 💾 Хранение данных

- **GitHub Gist** — единственный источник истины (source of truth)
- `localStorage` используется **только** для:
  - очереди оффлайн-изменений
  - временного состояния синхронизации

---

## 📥 Импорт / 📤 Экспорт

- Доступен локальный импорт и экспорт данных
- Используются как **резервное копирование**
- Импорт:
  - требует подтверждения
  - не отправляется в облако автоматически
- Экспорт:
  - создаёт JSON-файл с полным состоянием бюджета

---

## 🔐 Безопасность

- Авторизация и регистрация возможны **только онлайн**
- Пароли не сохраняются в `localStorage`
- Пароли надежно защищены шифрованием, и они не хранятся на сервере
- Все критические проверки выполняются на стороне Worker
